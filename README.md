# Equipment Management System

A full-stack Equipment Management System built using React, Express.js, Node.js, and MongoDB. The application helps manage equipment inventory, users, and equipment allocation through an admin-controlled system.

## Features

* Admin Authentication
* User Authentication
* Equipment Management
* Category Management
* User Management
* Equipment Allocation and Tracking
* Dashboard and Reports

## Tech Stack

### Frontend

* React
* Bootstrap
* Axios

### Backend

* Node.js
* Express.js
* MongoDB
* JWT Authentication

## Installation

### Clone Repository

git clone <repository-url>

### Backend Setup

cd express
npm install

### Frontend Setup

cd react
npm install

### Environment Variables

Create a .env file inside the express folder and add:

JWT_SECRET=

### Run Backend

npm start

### Run Frontend

npm start
